# Option 4: Tokyo + Hokuriku + Alps

## Shape

11 days / 10 nights.

Number of travel days: 5

- Day 1: arrive in Tokyo
- Day 4: Tokyo -> Kanazawa
- Day 7: Kanazawa -> Takayama
- Day 9: Takayama -> Matsumoto / Kamikochi side
- Day 11: return / departure buffer

- Tokyo: 3 nights
- Kanazawa: 3 nights
- Takayama: 2 nights
- Matsumoto / Kamikochi side: 2 nights

Suggested flights:

- SFO to Tokyo nonstop
- return from Tokyo after coming back by shinkansen, or use an open jaw if pricing is better

Transit overview:

- SFO -> Tokyo keeps the long-haul start simple
- Tokyo -> Kanazawa is a clean shinkansen move
- Kanazawa -> Takayama is the first more regional transfer and should be treated as a travel block
- Takayama -> Matsumoto / Kamikochi is the most coordination-heavy overland move in this option
- return via Tokyo or another major rail-connected city

## Best timing

June works well.

## Section highlights

### Tokyo

Overview:

- landing zone
- city autonomy
- easiest food and shopping section

Activities:

- [Tokyo destination overview](https://www.japan.travel/en/destinations/kanto/tokyo/)
- [Kiyosumi Garden](https://www.japan.travel/en/spot/1700/)
- [Shimokitazawa](https://www.japan.travel/en/spot/2108/)
- Daikanyama / Nakameguro wandering
- one strong dinner night

### Kanazawa

Overview:

- polished regional city
- easiest part of the route
- lower-stakes regional base

Activities:

- [Kanazawa overview](https://www.japan.travel/en/destinations/hokuriku-shinetsu/ishikawa/kanazawa-and-around/)
- [Kenrokuen Garden](https://www.japan.travel/en/spot/1399/)
- [Higashi Chaya District](https://www.japan.travel/en/spot/1405/)
- [Omicho Market](https://www.japan.travel/en/spot/1398/)
- craft / design shopping
- museum block
- slower cafe day

### Takayama + Matsumoto / Kamikochi

Overview:

- mountain-and-town core
- strongest scenery section
- most distinctive part of the route

Activities:

- [Hida Takayama overview](https://www.japan.travel/en/destinations/tokai/gifu/takayama-and-around/)
- [Sanmachi Historic District](https://www.japan.travel/en/spot/152/)
- [Kamikochi](https://www.japan.travel/en/spot/1357/)
- [Matsumoto Castle](https://www.japan.travel/en/spot/1356/)
- scenic bus / rail views
- shared hike
- stronger active branch
- photography day
- ryokan dinner

## Candidate itinerary

### Day 1: Arrive in Tokyo

- check in
- easy neighborhood walk
- early dinner

### Day 2: Tokyo shared city day

- coffee-and-walk morning
- suggested area: Daikanyama / Nakameguro, or Kiyosumi-Shirakawa if coffee is the priority
- shared lunch
- split afternoon
  - Track A: shopping / vintage / cafes
  - Track B: gym / run / photo walk
  - Track C: free city wandering
- one strong dinner

### Day 3: Tokyo flexible day

- morning free
- optional thrift / shopping branch in [Shimokitazawa](https://www.japan.travel/en/spot/2108/)
- optional gym / run / photo walk branch
- easy dinner

### Day 4: Transfer to Kanazawa

- shinkansen from Tokyo
- check in
- early walk in or near [Omicho Market](https://www.japan.travel/en/spot/1398/) if arrival timing works
- easy evening walk

### Day 5: Shared Kanazawa day

- [Kenrokuen Garden](https://www.japan.travel/en/spot/1399/)
- [Higashi Chaya District](https://www.japan.travel/en/spot/1405/)
- market / craft / cafe stops
- low-stakes dinner

### Day 6: Split Kanazawa day

- Track A: museums / shops / slower town day
- Track B: bike / longer walking day
- Track C: photography / hotel downtime

### Day 7: Transfer to Takayama

- overland transfer
- [Sanmachi Historic District](https://www.japan.travel/en/spot/152/) wandering
- easy dinner

### Day 8: Takayama day

- shared town-and-scenery day
- optional longer active branch

### Day 9: Transfer to Matsumoto / Kamikochi side

- overland transfer
- check in
- easy evening

### Day 10: Shared mountain day

- [Kamikochi](https://www.japan.travel/en/spot/1357/) or [Matsumoto Castle](https://www.japan.travel/en/spot/1356/) depending on weather and energy
- scenic walks
- moderate hike
- stronger active branch if energy is good

### Day 11: Return / departure buffer

- keep this day clean
- use it as the route's logistics buffer
