# Norway 2026

Three route options for the group trip. Same group, same dates (June 9–18 together, then split), different shapes. Each one is designed so P&L can get to Lofoten cleanly and everyone else can scatter.

---

## The three options

### [Option A — Out and Back from Bergen](option-a-bergen-out-and-back.md)
**Bergen → Sognefjord → Hardangerfjord → Bergen**

The simplest logistics. Fly into Bergen, loop through two fjord systems, return to Bergen and split. You get Trolltunga, Flam Railway, and fjord kayaking. Everyone departs from Bergen, which has the best connections for every direction.

---

### [Option B — Bergen to Alesund](option-b-bergen-to-alesund.md)
**Bergen → Sognefjord → Geirangerfjord → Alesund**

The maximum-scenery road trip. One-way north through Norway's two most iconic fjords, ending in the Art Nouveau coastal town of Alesund. You get Geirangerfjord and potentially Trollstigen, which Option A skips. The tradeoff: splitting from Alesund means longer connections for P&L to reach Lofoten, and fewer flight options for everyone else.

---

### [Option C — Alesund to Bergen](option-c-alesund-to-bergen.md)
**Alesund → Geirangerfjord → Sognefjord → Bergen**

Same route as B in reverse. You take the longer connection up front (flying into Alesund) when the group is together and fresh, then split from Bergen where connections are best. P&L get the easiest Lofoten flights. The tradeoff: everyone's inbound journey is 2–4 hours longer.

## Quick comparison

| | Option A | Option B | Option C |
|---|---|---|---|
| Fly into | Bergen | Bergen | Alesund |
| Split from | Bergen | Alesund | Bergen |
| Geirangerfjord | No | Yes | Yes |
| Trolltunga | Yes | No | No |
| Trollstigen | No | Maybe (if open) | Maybe (if open) |
| P&L → Lofoten ease | Best | Hardest | Best |
| Inbound flight ease | Best | Best | Hardest |
| Outbound flight ease | Best | Hardest | Best |
| Total driving | Least | Most | Most |
| Backtracking | Half-day return | None | None |
